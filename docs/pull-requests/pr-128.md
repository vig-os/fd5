## Summary
Phase 6 ingest layer with:
- `fd5.ingest._base`: Loader protocol, `hash_source_files()`, `discover_loaders()`
- `fd5.ingest.raw`: `ingest_array()`, `ingest_binary()`, `RawLoader` for numpy arrays
- `fd5.ingest.csv`: `CsvLoader` for CSV/TSV tabular data (spectrum, calibration, device_data)
- `fd5.ingest.nifti`: `NiftiLoader` for NIfTI-1/NIfTI-2 volumes (.nii, .nii.gz)
- `fd5.ingest.metadata`: RO-Crate and DataCite metadata import

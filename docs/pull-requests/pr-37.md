# [PR 37](https://github.com/vig-os/fd5/pull/37) feat(provenance): implement provenance group writers

## Summary

- Implement `fd5.provenance` module with three public functions: `write_sources`, `write_original_files`, and `write_ingest`
- `write_sources` creates `sources/` group with per-source sub-groups containing metadata attrs (`id`, `product`, `file`, `content_hash`, `role`, `description`) and HDF5 external links using relative paths
- `write_original_files` creates `provenance/original_files` compound dataset with `(path, sha256, size_bytes)` columns
- `write_ingest` creates `provenance/ingest/` group with `tool`, `tool_version`, `timestamp`, and `description` attrs
- All writers delegate to `fd5.h5io.dict_to_h5` for attribute writing, enforce write-once semantics, and achieve 100% test coverage (25 tests)

## Test plan

- [x] `write_sources` creates `sources/` group with description attr

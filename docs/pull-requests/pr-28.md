# [PR 28](https://github.com/vig-os/fd5/pull/28) feat(naming): implement generate_filename utility

## Summary

- Add `fd5.naming` module with `generate_filename(product, id_hash, timestamp, descriptors)` following the `YYYY-MM-DD_HH-MM-SS_<product>-<id>_<descriptors>.h5` convention
- Truncate `id_hash` to first 8 hex chars (strips `sha256:` prefix if present)
- Omit datetime prefix when `timestamp` is `None` (for simulations, synthetic data, calibration)
- 100% test coverage with 9 tests covering all acceptance criteria
